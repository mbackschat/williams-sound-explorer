Williams Sound Explorer

This explorer emulates the original Williams arcade sound ROMs (Defender 1980, Stargate 1981, Robotron 2084 1982). Those ROM images are © Williams Electronics and are not distributed here — supply your own copy below.

🔒 Files are read in your browser only and stored locally (IndexedDB) — nothing is uploaded to any server. You need the sound ROM for each game (from a MAME romset you're licensed to use, or dumped from your own board). You can also build them from source with tools/build_roms.sh. Drop a file on a slot — the app checks it and shows its SHA-1.

How the scrubber works

Click Scrub, then a yellow marker to jump to that sound's start. Speed is frozen by default — press ⏩ to play, ⏪ for reverse. Gray ticks mark where each sound ends; the gap after a gray tick is the silence between sounds.
Loop modes: off = stop at end · range = wrap whole recording · segment = wrap within current sound (clicking a marker switches to this mode for repeat-listening).

Spectrogram

Scrolling FFT of the post-volume signal — low freq bottom, high top. LITE's LFSR noise is a broadband upward sweep.

RAM heatmap zero-page write activity

Each cell of the sound CPU's 128-byte zero-page RAM. Red intensity fades over ~1 second since last write. Watching which cells light up while a sound plays is the cheapest way to find an engine's working set.
